# Proyecto Web - Módulos (Home, Planes, Cotización)

Sitio web desarrollado con **HTML, CSS y JavaScript** (Frontend) con módulos de:

- **Home** (Inicio)
- **Planes**
- **Cotización**

📌 **Despliegue:** Vercel  
📌 **Base de datos (planeada):** MySQL (documentada, aún no integrada desde el frontend)

---

- **Rutas:**
  - Home: `/`

---

## 🧩 Módulos del sistema

### 1) Home (Inicio)

- Archivo: `public/index.html`
- Función: Presenta la página principal y navegación hacia los demás módulos.

### 2) Planes

- Archivo: `public/planes.html`
- Función: Muestra los planes disponibles (estáticos por ahora).

### 3) Cotización

- Archivo: `public/cotizacion.html`
- Función: Formulario para capturar datos de cotización y mostrar confirmación en pantalla (por ahora).

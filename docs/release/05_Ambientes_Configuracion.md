# 05. Ambientes y Configuraciones

**Proyecto:** Sitio Web (Frontend) - Módulos Home, Planes y Cotización  
**Autor:** Salgado González, Deivid Yoset  
**Fecha:** 05/05/2026  
**Versión:** 1.0

## 1. Ambiente de Desarrollo

- Equipo: PC local.
- Editor recomendado: Visual Studio Code.
- Ejecución local:
  - Abrir `index.html` o
  - Usar extensión Live Server.

## 2. Ambiente de Pruebas

- Navegadores sugeridos: Chrome / Edge / Firefox.
- Validaciones:
  - navegación
  - responsividad
  - carga de imágenes y estilos
  - validación de formulario

## 3. Ambiente de Producción (Vercel)

- Despliegue automático conectado al repositorio Git.
- Nota: El entorno de Vercel suele ser Linux, por lo que:
  - **Mayúsculas/minúsculas importan** en nombres de archivos.
  - Ej: `Logo.png` NO es lo mismo que `logo.png`.

## 4. Convenciones para recursos (imágenes)

- Guardar en: `assets/img/`
- Referenciar con ruta absoluta:
  - `/assets/img/archivo.png`
- Evitar espacios y caracteres especiales en nombres.
